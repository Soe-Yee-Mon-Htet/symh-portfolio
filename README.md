## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Soe-Yee-Mon-Htet/symh-portfolio.git
```

Navigate to the project directory:

```bash
cd symh-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
